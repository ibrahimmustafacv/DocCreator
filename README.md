# DocCreator - محرر الوثائق البسيط

## نظرة عامة
محرر وثائق بسيط يدعم اللغة العربية والإنجليزية مع إمكانية التصدير إلى PDF و Word. يتميز بواجهة حديثة وسهلة الاستخدام.

## المميزات

### ✨ المميزات الرئيسية
- **دعم ثنائي اللغة**: العربية والإنجليزية مع دعم كامل للنصوص من اليمين لليسار
- **تصدير متعدد**: إمكانية تصدير الوثائق إلى PDF و Word
- **رفع الصور**: إمكانية إضافة الصور مع السحب والإفلات
- **معاينة مباشرة**: معاينة الوثيقة في الوقت الفعلي
- **تصميم متجاوب**: يعمل على جميع الأجهزة

### 🎨 واجهة المستخدم
- تصميم حديث وبسيط
- دعم الوضع المظلم
- واجهة متجاوبة للهاتف والكمبيوتر
- أزرار تفاعلية للتنقل

### 📱 التوافق
- جميع المتصفحات الحديثة
- الهواتف الذكية والأجهزة اللوحية
- سطح المكتب

## طريقة الاستخدام

### 1. الاستخدام المباشر
افتح ملف `index.html` في أي متصفح للاستخدام المباشر.

### 2. التطوير المحلي
```bash
# تثبيت المتطلبات
npm install

# تشغيل الخادم المحلي
npm run dev

# بناء الموقع للإنتاج
npm run build
```

### 3. الرفع على GitHub Pages
1. ارفع جميع الملفات إلى مستودع GitHub
2. فعل GitHub Pages من إعدادات المستودع
3. اختر الفرع الرئيسي كمصدر

## التقنيات المستخدمة

### Frontend
- **React 18** - مكتبة واجهة المستخدم
- **TypeScript** - لغة البرمجة مع الأنواع
- **Tailwind CSS** - إطار عمل CSS
- **Vite** - أداة البناء والتطوير

### Backend
- **Node.js** - بيئة تشغيل JavaScript
- **Express** - إطار عمل الخادم
- **Drizzle ORM** - أداة إدارة قاعدة البيانات

### مكتبات التصدير
- **jsPDF** - تصدير PDF
- **html2canvas** - تحويل HTML إلى صورة
- **docx** - تصدير Word
- **FileSaver.js** - تحميل الملفات

## التخصيص

### تغيير الألوان
عدل ملف `tailwind.config.ts` لتغيير الألوان:

```typescript
colors: {
  primary: '#3b82f6',
  secondary: '#64748b',
  // إضافة ألوان جديدة
}
```

### إضافة لغات جديدة
1. أضف اللغة الجديدة في `LanguageType`
2. أضف الترجمات في المكونات
3. أضف دعم الاتجاه RTL/LTR
